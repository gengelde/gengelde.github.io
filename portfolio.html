Project "Sinker" (GitHub Repository)

Description: Sinker is a game of strategy where players are to guess the locations of hidden ships in a 10x10 grid. With a total of 10 ships each of varying lengths and 80 attempts, the player must find and sink all 31 ship pieces.

Project "Hexmatch" (GitHub Repository)

Description: Hexmatch is a memory and reaction time testing game implemented on a Raspberry Pi 4. The game challenges players to match randomly generated hexadecimal symbols/characters displayed on the TM1638 module with a target symbol on the 1-bit 7-segment LED display. The objective is to successfully complete 10 matches in a row.
